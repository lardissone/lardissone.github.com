Actualización automática de Wordpress

Fri 07 September 2007
By Leandro Ardissone

In General.

tags: wordpress

Ya hablé de un plugin para Wordpress que permite instalar plugins (valga la redundancia) automáticamente sin tener que andar subiendo los archivos por FTP manualmente. En este caso, un plugin que permite actualizar Wordpress a la última versión sin tener que meter mano en nada.

El plugin chequea que tu sistema sea compatible para correr el upgrade automáticamente y luego realiza backup de los archivos y base de datos. Luego se instalan los nuevos archivos y listo!
