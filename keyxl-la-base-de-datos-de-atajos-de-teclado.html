KeyXL, la base de datos de atajos de teclado

Fri 14 September 2007
By Leandro Ardissone

In General.

tags: sitiosservicios


KeyXL es un sitio donde se recopilan todos (o casi todos) los atajos de teclado (si, es horrible en español Keyboard Shortcuts).
Hay para todo tipo de programas y sistemas operativos: Microsoft Word/Excel/Access/Powerpoint, Internet Explorer, Mozila Firefox, Photoshop, Corel, de lo que busques.

Y uno puede enviar nuevos atajos enviandoselos por mail al autor.
