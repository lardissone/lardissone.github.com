Invitación para Measure Map

Fri 11 November 2005
By Leandro Ardissone

In General.

tags: internet

Fuí invitado para testear Measure Map. Por el momento parece bastante interesante.
Un sistema especialmente diseñado para estadisticas de blogs, y requiere una configuración especial para cada programa de publicación.

Produce estadisticas por post y comentarios, bien detallados.

Lo voy a probar unos días y les cuento que tal funciona.

Creo que a Mint le empieza a doler la oreja. (por no decir otra cosa).
