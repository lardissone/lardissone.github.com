jQuery UI, los componentes gráficos de jQuery

Mon 17 September 2007
By Leandro Ardissone

In Desarrollo.

tags: jqueryjavascript


jQuery lanzó una nueva librería para crear aplicaciones visuales más fácilmente que denominaron jQuery UI (por User Interface, claro). Consta de un aprovechamiento de las funciones que ya tenía jQuery en su librería y una demostración de las mismas.

La página tiene un diseño muy bonito y los ejemplos están muy claros. Además incluye unos widgets que antes se encontraban como plugins externos de jQuery, como ser el accordion (para crear menus o listados auto expandibles), calendar (para ingresar fechas en formularios de manera rápida), dialog (para crear ventanas/popups de aviso o interacción), slider (para barras de deslizamiento), table (para tablas ordenables via javascript) y tabs (para navegación por solapas). Estos son los oficiales por el momento, pero ya se encuentran en desarrollo los siguientes: Menu, Form Controls, Tree, Splitter, Rich Text Editor, y Uploader, que se van a ir agregando a medida que se terminen de testear.

También agregaron algunos efectos nuevos como: magnifier (efecto lupa) y shadow (efecto de sombra).

La verdad que un laburo increible.

Vamos jQuery todavía! (Cómo te quedó el ojo prototype.js?) ;)
