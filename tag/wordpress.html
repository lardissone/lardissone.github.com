Instala plugins o temas de Wordpress directo desde Firefox. Sin usar FTP

Tue 28 August 2007
By Leandro Ardissone

In General.

tags: wordpressfirefox

Hace unos días encontré que existía un plugin para Wordpress para automatizar la instalación de plugins y temas desde el mismo panel de administración de Wordpress, pero esta extensión para Firefox permite hacerlo desde el mismo navegador.

Una vez que encontrás una extensión que te interesa, simplemente das click derecho sobre el link al archivo se le indica si es un tema o una extensión y listo, luego solo es cuestión de entrar a Wordpress a configurarlo.
