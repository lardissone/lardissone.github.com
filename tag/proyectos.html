WP Plugins

He aquí una colección de los plugins para Wordpress que voy haciendo.

Wishlists

Version 0.2b

Update: Guido, me hizo notar que traía conflictos con otros plugins que usen la librería SimplePie, así que quedó arreglado en esta versión.

Este plugin sirve para mostrar algunos productos de las listas de deseos de MetaWishlist o Wists.
Para demostrarle a quienes visitan tu blog que es lo que te podrían regalar.

MetaWishlist y Wists son sitios de listas de 'productos deseados' sociales, donde la gente puede compartir con otras personas sus listas. Además proveen herramientas que se adaptan al propio navegador para agregar productos desde cualquier sitio que estés navegando.

Por el momento, solo incluyo estos dos sitios, pero la idea es incluir la lista de Amazon, ThinkGeek, entre otros.

Instalación:
Copiar toda la estructura de archivos incluida en el .zip dentro de la carpeta de plugins de tu instalación de Wordpress (/wp-content/plugins/-acá-) de manera que te va a quedar algo así:

/wp-content/  
/wp-content/cache/  
/wp-content/wishlists.php  
/wp-content/options_wishlists.php

Y darle permisos de escritura a la carpeta 'cache' (chmod 0755 o sino te funciona chmod 0777 pero es menos seguro).

Luego ir a la administración de Plugins dentro del panel de administración de Wordpress y activar el plugin 'Wishlists'.

Una vez hecho esto, decidirás donde quieras incluir el plugin, es decir, en que parte del template.
Y para agregarlo debes poner el siguiente código:

show_wishlists ($cant, $cache_time)

Los dos parametros son opcionales:
$cant indica cuantos items de cada sitio querés mostrar, por default son 5.
$cache_time cada cuanto tiempo querés que se refresquen los feeds, esto es para acelerar la carga del sitio y reducir las comunicaciones con los sitios de las listas. Por default es 30 minutos.

El formato de salida del plugin es (por cada servicio):

<ul>  
<li class="wish-title"><a href="...">Nombre del Servicio</a></li>  
<li class="wish-item"><a href="...">item..</a></li>  
<li class="wish-item"><a href="...">item..</a></li>  
<li class="wish-item"><a href="...">item..</a></li>  
...  
</ul>
