3 razones para porqué usar jQuery

Fri 01 June 2007
By Leandro Ardissone

In Desarrollo.

tags: javascriptjquery

Desde el principio que me volví fanático de la maravillosa librería de javascript jQuery. Lo cual me volvió un tanto vago a la hora de programar javascript sin usar la ayuda de esta librería.

En este sitio explican cuales son las principales tres razones de porque utilizar jQuery.

  1. Selectores: jQuery soporta muchos selectores. Y cuando digo un muchos, realmente quiero decir muchos. Primero, están los selectores CSS. Solo debés aplicar lo que sabes de CSS y podes usar muchos selectores complejos. Segundo, están los selectores XPath. Cuando era joven, escribí mucho XSL. Y cuando digo mucho, estoy diciendo mucho. XSL hace un uso intensivo de XPath. Porque XHTML es básicamente XML, por esto podemos usar XPath para navegar fácilmente a través del DOM. El problema era que antes que jQuery, nada podía hacerlo. Ahora podemos! En comparación, prototype solo tiene el comando $('ID') para seleccionar un elemento. Ouch!
  2. Los Atributos: Es fácil para agregar o quitar atributos a los elementos. Uso esto cuando quiero crear nuevos elementos o cuando necesito agregarle/quitarle una clase CSS a un elemento. Lo más interesante es que podes encadenar los comandos y va a funcionar. $("div").attr("title", "Esto es un div").addClass("newClass");
    Podes encadenar la cantidad que quieras.
  3. Ajax y JSON: Otra vez, Ajax es ahora tan fácil como 1,2,3. Escribí un artículo sobre JSON y pensé, no existe una alternativa real a JSON. jQuery te ayuda con esto.
    Con prototype esto es un poco mas complejo.
$.getJSON("http://www.thesite.com/thejson",
function(json){
    alert("I received the json and put it in the json var : " + json.toString());
}
);
