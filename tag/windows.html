Acceso directo para crear nuevas carpetas

Thu 13 September 2007
By Leandro Ardissone

In General.

tags: softwaretipswindows

Desde siempre tuve el mismo dilema, como no van a asignarle una combinación de teclas para crear una carpeta en Windows (XP al menos), es una locura. Una de las tareas mas frecuentes en el Explorador de Windows es crear carpetas y no le asignaron ninguna!

La única forma posible de hacerlo es en el explorador de Windows apretar la siguiente seguidilla de teclas (que corresponde a abrir el menu File -> New -> Folder en Windows en Inglés): Alt -> F -> W -> F. En Español sería: Alt -> A -> N -> C.
Todo muy lindo pero para eso hago botón derecho con el mouse. Lo ideal sería una combinación de dos teclas simultáneamente.

Para eso HowTo Geek nos recomienda el uso de bxNewFolder, que asigna un acceso directo a la tecla F12, de manera que cada vez que presionemos dicha tecla nos aparezca la opción de crear la carpeta. No es la mejor solución, pero.. algo es algo.

Link: bxNewFolder
