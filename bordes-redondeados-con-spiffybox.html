Bordes redondeados con SpiffyBox

Thu 11 October 2007
By Leandro Ardissone

In General.

tags: sitiosservicios


Con el nombre gracioso de esta aplicación podemos crear bordes redondeados para usar en sitios web.
Ingresamos los colores que queremos y el sistema automáticamente nos devuelve la imagen y el código CSS para armar nuestros recuadros/ventanas/loquesea con bordes redondeados de manera simple.
