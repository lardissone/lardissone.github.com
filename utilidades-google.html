Utilidades Google

Mon 20 September 2004
By Leandro Ardissone

In General.

tags: google

Este es un rejunte, por decirlo de alguna forma, de aplicaciones online basadas en Google y sus APIs. La mayoría de estos links no están asociados con Google, y son experimentos. Muchos de ellos son inútiles pero son divertidos.
En algún momento y con mas tiempo voy a categorizarlos.

Google API Proximity Search:
-Este script en Perl usa la API de Google para buscar la distancia entre dos términos en una misma página.

Google API Relation Browsing Outliner:
-Permite ingresar una URL y buscar sitios que estén relacionados o linkeen al mismo.

Google API Web Search by Host:
-Este script permite buscar por un término y devuelve los dominios que lo contienen.

Google Dance Tool:
-Es una herramienta que permite al webmaster determinar cuando Google está corriendo sus arañas por la Internet. Es muy útil para saber cuando un sitio será indexado o rankeará en Google.

GooFresh:
-Google permite hacer una búsqueda por sitios que fueron agregados a su indice [o base de datos] un día determinado, pero la sintaxis de búsqueda suele ser medio tediosa, ésta herramienta permite hacerlo de manera fácil.

Search Sinker:
-Google basa sus resultados en el numero de veces que uno repita una palabra buscada. Esta herramienta permite hacer una búsqueda utilizando varias keywords pero dándole mas énfasis a una en particular.

Wumwum:
-Busca quienes están linkeando a las páginas o sitios que se le especifique, pero puede excluir dominios que no se desean ver. Por ejemplo para tomar estadísticas.

G-Metrics:
-Muestra el googlecount de un sitio o keyword.
